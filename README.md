# FUTURE_PE_02 – Prompt Engineering System for Local Business Growth

## 📌 Project Overview
This project focuses on building a structured prompt engineering system to generate high-quality marketing content for a local business.

The selected business is:
**Sri Nookambika Tiffin Center**

This project builds on Task 1 and focuses on:
- Creating reusable AI prompts
- Generating marketing content
- Improving business reach using AI-driven outputs

---

## 🎯 Objectives
- Design powerful prompts for content generation
- Generate scripts, hooks, captions, and CTAs
- Improve engagement for social media platforms
- Maintain consistency in brand voice

---

## 🧠 Business Context
Sri Nookambika Tiffin Center is a local food business offering:
- Idli
- Dosa
- Poori
- Upma
- Traditional South Indian breakfast

Target audience:
- Students
- Office workers
- Families
